HTML (HyperText Markup Language) to język znaczników służący do tworzenia stron internetowych. Składa się ze znaczników, które definiują strukturę i zawartość strony, takich jak nagłówki, akapity, listy, obrazy, linki i wiele innych. HTML jest podstawą każdej strony internetowej i jest interpretowany przez przeglądarki internetowe, które wyświetlają zawartość strony użytkownikowi.

CSS (Cascading Style Sheets) to język służący do określania wyglądu elementów HTML. Pozwala na dostosowanie kolorów, czcionek, układu strony, animacji i wielu innych aspektów wizualnych. CSS jest oddzielony od struktury HTML, co ułatwia zarządzanie wyglądem strony i umożliwia stosowanie tych samych stylów do wielu stron.

JavaScript to język programowania, który dodaje interaktywność do stron internetowych. Pozwala na tworzenie dynamicznych elementów, takich jak menu rozwijane, formularze, animacje, interakcje z użytkownikiem i wiele innych. JavaScript jest wykonywany po stronie klienta, co oznacza, że kod jest przetwarzany przez przeglądarkę internetową użytkownika.
